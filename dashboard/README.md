# Day7-Day8 Streamlit 数据看板

## 本层位置

看板是数据链路的展示层。它读取已经通过校验的 DWS 和 DM 结果，把月度 KPI、趋势、平台、类目、企业和地区数据放到同一页面中。

看板不修改数据库，不重新清洗源数据，也不在页面运行时重新定义指标公式。页面中的数字应当能够回溯到 DWS 或 DM，数据问题应返回上游修正，而不是只在界面中隐藏。

## 上游依赖

| 页面内容 | 主要数据源 | 上游职责 |
|---|---|---|
| KPI、物流估算、环比、同比 | `dm_monthly_business_metrics_2026_gj` | 提供已验证的业务公式 |
| 企业销售排行 | `dm_enterprise_sales_rank_2026_gj` | 提供每月、每平台企业排名 |
| 类目 TOP10 | `dws_category_month_summary_2026_gj` | 提供完整类目汇总，支持所选平台重排 |
| 省市筛选、区县排行、地图 | `dws_region_month_summary_v2_final_2026_gj` | 提供证据恢复后的区域层级和地区范围 |

只有这些上游表完成刷新和校验后，看板结果才具有业务意义。

## 页面

### 1：确定当前分析范围

页面先读取可用月份和平台，形成全局筛选条件。

月份采用单点滑块。滑块停在哪个月，KPI、平台构成、类目、企业和区域查询就只读取该月，不再把起止月份之间的数据累加。默认月份为当前数据范围中的最后一个月。零售额和物流趋势固定保留 202601-202606 全周期，作为所选月份的变化背景。

平台支持多选，所选平台影响全部主要结果。这样同一个页面可以查看全平台，也可以缩小到一个或多个平台。

### 2：展示核心业务结果

页面首先展示零售额、销量、快递收入估算、快递量估算和活跃平台店铺数。

这些指标直接来自 DM。顶部零售额环比会单独读取上月作为基期，因此即使月份筛选只选一个月，仍可以正确显示环比。

### 3：解释平台和类目构成

平台分布展示所选月份、所选平台的销售额构成；类目 TOP10 从完整类目汇总中重新聚合和排序。

类目图只展示有证据的已分类类目，标题为“类目销售额 TOP10（不含未分类）”。“未分类”不占用排名名额，但仍保留在完整类目汇总和占比分母中；图表旁同时展示已分类销售额占比、未分类销售额和未分类占比。这些指标随月份和平台筛选同步变化。

看板不执行比例分配。类目恢复由上游 DWS 根据真实类目、唯一类目 ID 和唯一店铺历史完成，仍无证据的数据继续显示为未分类。

### 4：提供企业排行

企业排行读取 DM 已计算的各平台 TOP20。选择多个平台时，页面分别展示这些平台的排名，不在运行时重新扫描完整企业排行表。

这一设计保证页面响应速度稳定，也保持排名口径与 DM 一致。当前排行是平台内排行，不是多个平台合并后的全平台排行。

### 5：提供地区下钻

省份和城市筛选只读取 `region_scope='DOMESTIC'` 的标准行政区，作用于区域选项、中国大陆区县 TOP15 和地图。境外、港澳台和范围未知记录不混入中国行政区排行，但仍保留在全平台总销售额中。

区县图只展示已经定位到具体区县的国内数据。图表旁同时展示中国大陆已定位、大陆待识别、境外或地区不适用、范围未知四类销售额和占比，使被排除的未知值仍然可见且可对账。

上游只使用行政层级、唯一店铺历史、完全地址和至少两个一致地址层级作为恢复证据。没有证据的记录继续保持未知，页面不执行比例分配，也不把境外销售额伪装成中国地区。

### 6：展示数据状态和空结果

当销售额有效店铺覆盖不足时，页面显示覆盖率提醒；当基期缺失或为 0 时，同比显示无可用同期基期；当筛选没有结果时，页面显示明确空状态。

这些提示把 DWD 和 DM 中的质量状态传递到展示层，防止用户把缺失数据误解为真实 0。

## 联调优化与 Bug 修复

| 页面问题 | 调整后的职责 | 验证结果 |
|---|---|---|
| 月份滑块有两个端点，容易被理解为累计区间 | 改成单点月份；业务构成和排行只查当前月，趋势保留上半年全周期，环比从趋势中的上月取得 | 切换 202601-202606 时单月指标变化，趋势口径保持稳定 |
| 省份和城市出现“云南/云南省、上海/上海市”等重复项 | 筛选选项只读取区域 V2 中已标准化的中国大陆名称 | 省市选项唯一，广东省到深圳市联动正常 |
| `Office Products`、`国际邮编` 等非行政区进入筛选 | 非行政文本不进入标准行政区，无法确认范围的数据保留质量分组 | 筛选列表不再出现非行政区文本 |
| 企业排行查询约 1200 万行导致超时或连接中断 | 页面读取 DM 已计算的每月、每平台 TOP20 | 页面不再运行企业全表重聚合 |
| “未分类”长期占据类目 TOP10 第一名 | 上游证据恢复类目；图表排除未分类并单独显示覆盖率 | 未分类仍参与总额对账，不占排名名额 |
| 区县排行出现“三未知”和境外混入 | 改读区域 final 表，只排名已确认的中国大陆区县 | TOP15 无未知、境外或“不适用”标签 |
| 长金额在指标卡被截断 | 调整指标字号、宽度和换行规则 | 桌面和窄屏均可读完整指标 |

区域过程表 `dws_region_month_summary_v2_2026_gj` 只保留构建和故障审计用途。看板配置必须指向 `dws_region_month_summary_v2_final_2026_gj`，避免把过程表中的重复全国行带入页面。

## 页面组成

- 月度零售额和销量指标；
- 物流收入与物流量指标；
- 平台销售额分布；
- 类目销售额 TOP10；
- 中国大陆区县销售额 TOP15及地区覆盖质量；
- 各平台企业 TOP20；
- 平台月度环比和同比；
- 省域销售分布地图。
