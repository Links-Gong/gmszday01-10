# DM 结果证据

## 目录作用

本目录保存业务指标、企业排行样本和公式校验结果。它证明 DM 不只是生成了数据，还保持了与 DWS、DWD 和 2025 基期的连续关系。

## 结果逻辑

### 1. 月度业务指标

月度结果展示每个月、每个平台的零售额、销量、物流估算、环比、同比、店铺数和数据状态。

它是看板 KPI 和平台月度变化的直接数据源，预期覆盖 24 个平台月分组。

### 2. 企业排行样本

企业样本用于检查平台内排名、企业名称、销售额、销量和店铺数是否合理，并证明看板读取的 TOP20 来自已经计算的 DM 排名。

企业排行全表约 1200 万行。验收和版本归档只需保存各月、各平台的 TOP 样本和汇总行数，不需要重复导出完整表。历史全量导出可保留为已有成果，但不应作为日常验证方式。

### 3. 物流公式验证

公式结果复算销售额的 15% 和销量的 53%，证明 DM 中的快递收入和快递量与固定业务公式一致。

### 4. 同比状态验证

同比结果同时展示当前金额、2025 同期金额、同比值和状态。它用于区分正常计算、缺失基期、空基期、0 基期和空当前值。

当前 2025 基期已完成，因此 202601-202606 的 24 个平台月同比均应为 `CALCULATED`。

### 5. 上下游对账

DM 与 DWS 的金额、销量和店铺数差异应为 0；企业唯一性检查应返回 0 条重复记录。

这一步证明看板使用的业务指标没有脱离上游主题汇总和店铺事实。

## 完成标志

- 月度指标覆盖 24 个平台月分组；
- 物流公式差异为 0；
- 同比公式和状态正确；
- DM 与 DWS 对账差异为 0；
- 企业粒度无重复且排行样本合理。

达到这些条件后，DM 可以作为看板的稳定业务数据源。
